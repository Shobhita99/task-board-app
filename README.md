# 📋 Smart Task Board Application

A full-stack task management application with a unique **Smart Priority Sorting System** built with Python Flask and vanilla JavaScript.

---

## ✨ Features

### Core Features
- ✅ **Add Tasks** - Create tasks with custom titles
- 🎯 **Priority Levels** - Assign High, Medium, or Low priority
- 🔄 **Smart Auto-Sorting** - Tasks automatically sorted by priority
- ✓ **Task Completion** - Toggle tasks as complete/incomplete
- ❌ **Delete Tasks** - Remove tasks with one click
- 📊 **Progress Tracking** - Real-time progress bar and statistics
- 📱 **Responsive Design** - Works on all devices

### 🌟 Unique Feature: Smart Priority Sorting System
- **Two-Level Sorting:** Incomplete tasks first, then by priority
- **Visual Priority Badges:** Color-coded (Red=High, Yellow=Medium, Gray=Low)
- **Dynamic Re-sorting:** Tasks automatically re-sort when completed
- **Consistent Logic:** Same sorting on both backend and frontend
